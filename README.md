Este projeto é um sistema web de controle financeiro pessoal desenvolvido em Java utilizando JSF (Jakarta Server Faces). O objetivo principal da aplicação é ajudar o usuário a organizar suas finanças de forma simples, segura e intuitiva, permitindo o registro de receitas, despesas, metas mensais e análises automáticas de desempenho financeiro. O sistema foi pensado para ser uma ferramenta de organização, sem qualquer integração com dados bancários ou de cartão de crédito, garantindo maior privacidade e segurança para o usuário.

A aplicação permite o cadastro obrigatório de usuários, com autenticação por e-mail e senha. Após o login, o usuário pode registrar suas receitas e despesas, classificando-as em categorias como alimentação, contas básicas, internet, celular, fatura de cartão, entre outras. Além das categorias padrão, o sistema também permite a criação de categorias personalizadas. O usuário pode cadastrar despesas recorrentes, definindo se elas devem ser registradas automaticamente ou apenas exibidas como notificações mensais.

O sistema oferece visualização dos dados por mês, possibilitando o acompanhamento do histórico financeiro. A cada período, o sistema calcula automaticamente o saldo, indicando se houve lucro ou prejuízo. Também é possível comparar o desempenho financeiro com meses anteriores, exibindo a evolução em percentuais e gerando insights automáticos, como aumentos ou reduções nos gastos. O usuário pode ainda definir metas mensais de lucro e verificar se elas foram atingidas com base no saldo calculado.
